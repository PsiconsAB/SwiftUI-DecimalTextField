# SwiftUI-DecimalTextField

A **robust**, **reusable** SwiftUI `TextField` for numeric input that:

- Accepts **both `.` and `,`** as decimal separators  
- Uses **`NumberFormatter`** for full control  
- **Blocks invalid input** (letters, symbols)  
- **Formats only when editing ends**  
- **Prevents accidental erase** on invalid input  
- **No external dependencies** â€” pure SwiftUI  

Perfect for **scientific**, **financial**, or **international** apps.
